# Store Rating System

## Overview

Store Rating System is a web application where users can view stores and submit ratings. Admins can manage users and stores, while store owners can view ratings and store statistics.

## Technologies Used

* React.js
* Node.js
* Express.js
* MySQL
* Sequelize ORM
* JWT Authentication
* Bootstrap

## Features

### Admin

* View Users
* View Stores
* Add Users
* Add Stores
* Dashboard Statistics

### User

* Register/Login
* View Stores
* Submit Ratings
* Update Ratings

### Store Owner

* Login
* View Store Ratings
* View Average Rating

## Installation

### Backend

```bash
cd backend
npm install
node app.js
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Database

Create a MySQL database and update database credentials in the backend configuration.

## Default URLs

Frontend:
http://localhost:5173
